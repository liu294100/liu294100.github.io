代理分流检测

通过同时请求国内和国外的 API 检测你的网络是否启用了代理分流（Split Tunnel），并分析分流规则是否生效。

原理：如果你使用了代理分流，访问国内网站时会走直连（显示真实 IP），访问国外网站时会走代理（显示代理 IP）。两者 IP 不同则表示分流生效。
