网站分流测试

当使用了 Split Tunnel VPN 或分流代理（如 Clash）时，这个工具可以查看你访问不同网站时所使用的出口 IP 地址，用于测试分流规则是否生效。

原理：通过让你的浏览器分别请求不同域名下的 IP 检测接口，获取你访问每个域名时实际使用的出口 IP。不同的 Clash 规则会让不同域名走不同的代理节点，因此出口 IP 会不同。
